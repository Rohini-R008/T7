# Laundry Services Responsive Webpage

## Description

This project implements a responsive hero section for a laundry service web application using HTML and CSS. The page includes a navigation bar and a hero section with text and an image. Media queries are used to make the layout responsive for desktop, tablet, and mobile devices.

## Features

* Responsive navigation bar
* Hero section with heading, description, and button
* Image displayed alongside the text
* Layout adapts for desktop, tablet, and mobile screens
* Flexbox used for responsive alignment
* Media queries used for different screen sizes

## Files Included

* **index.html** – Contains the structure of the webpage
* **index.css** – Contains the styling and responsive media queries

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open **index.html** in any web browser.
4. Resize the browser window or use developer tools to test responsiveness.

## Technologies Used

* HTML
* CSS
